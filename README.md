# sublime-setup

The easiest way to install is using [Package Control](https://packagecontrol.io/)

Installation

1. Open Command Palette ```Ctrl + Shft + p```.
2. Type ```pcip```
3. Type your desired them

Activation

4. ```Preferences``` > ```Package Settings``` > ```Boxy Theme``` > ```Activation```
